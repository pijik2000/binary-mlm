== Description ==

The plugin enables you to run a full blown Binary MLM Network inside of Wordpress.

Features:

1. New members can register at the front end directly by specifying Sponsor Username
2. Existing members can register new members by either opening the Registration Form or Clicking on an empty node in the Binary Tree.
3. Facility to mark members as paid / unpaid from the admin
4. The tree view works perfectly in all browsers and OS.
5. Specify base currency in the admin
6. Specify eligibility criteria in the admin
7. Specify payplan (commission + bonus) details in the admin
8. Run payouts manually from the admin.
9. Each member is able to see complete details for commission and bonus earned during one payout cycle.
10. Exhaustive help text on each settings screen.

Check out Binary MLM Pro and Binary MLM e-Commerce at <a href="http://wordpressmlm.com" target="_blank">http://wordpressmlm.com</a>

If you have any support issue related to Binary MLM, please write to us at <a href="mailto:support@wordpressmlm.com">support@wordpressmlm.com</a>

== Installation ==

1. Place the 'binary-mlm' folder in your '/wp-content/plugins/' directory.
2. Activate Binary MLM.
3. A new menu item would be created in the admin menu called Binary MLM.

== Frequently asked questions ==

= Does the plugin support multiple currencies? =

The plugin gives you the option to choose the base currency for your network. However, once chosen the currency cannot be changed.
